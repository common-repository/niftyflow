Create multilingual price quote, ROI, and finance calculators with drag & drop. Easy to embed on any website. No developer or coding skills required.

== Description ==

Niftyflow is a drag-and-drop form and calculator builder that enables you to easily create highly customizable form and calculator widgets.
This plugin allows you to easily embed widgets created with our builder into your WordPress website using shortcodes.

== Demo ==
Try a demo of our easy calculator builder for free [on our website](https://niftyflow.rocks) - instant access, no signup required.

== Requirements ==
The plugin requires a niftyflow account. You can sign up for free at [niftyflow.rocks](https://niftyflow.rocks).

== Installation ==

1. Install using the WordPress built-in Plugin installer, or extract the zip file and drop the contents into the wp-content/plugins/ directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Before proceeding, make sure you've prepared a form or calculator for embedding. If you haven't created one yet, you can do so at [niftyflow.rocks](https://niftyflow.rocks).
4. In your form or calculator widget on niftyflow navigate to Share > WordPress Plugin, and copy the shortcode.
5. You can now use the shortcode in any post/page and with page builders, including Gutenberg and Elementor."

== Screenshots ==

1. Copy the shortcode from your calculator or form widget from the niftyflow editor
2. Add a "Shortcode" block in the WordPress editor
3. Paste the widget shortcode you copied earlier into the "Shortcode" block
4. After saving your changes your calculator or form widget will be visible on the preview page
